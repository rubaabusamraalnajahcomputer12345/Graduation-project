## üöÄ Quick Start

### Prerequisites
- Flutter SDK 3.7.2+
- Node.js 18+
- MongoDB 5.0+

### 1. Clone Repository
```bash
git clone https://github.com/your-username/hidaya.git
cd hidaya
```

### 2. Backend Setup
```bash
cd backend
npm install
npm start  # Server runs on http://localhost:3000
```

### 3. Frontend Setup
```bash
cd frontend
flutter pub get
flutter run -d chrome  # For web development
```
